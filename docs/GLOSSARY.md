# noted Glossary

## Note

A short text entry stored by the `noted` CLI.

## Note ID

A numeric identifier assigned to each saved note, starting at `1` and increasing for every new note.

## `add` Command

The command that saves a new note, for example `python noted.py add "buy milk"`.

## `list` Command

The command that prints all saved notes, or `no notes yet` when the note database is empty.

## `NOTED_DB`

An environment variable that overrides the default storage path of `~/.noted.json`.
